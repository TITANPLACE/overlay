Overlay Instructions
  1. Install Tampermonkey (Chrome, Edge or Opera), or Violentmonkey (Firefox). Android users can use Kiwi Browser and install the script as usual.
  2. Go to this webpage and install the script by pressing "Install".
  3. Refresh r/place for the script to take effect. If everything went right, you'll see squares inside of squares around those coordinates. The color of the inside square is what color the outside square needs to be.
